Déploiement : https://loqui.free.nf/

Projet de création d’une application de messagerie web en HTML, CSS et PHP.
Elle inclut l’inscription, la connexion, un chat général, des messages privés, l’envoi de fichiers et la gestion d’une base de données.
Ce projet universitaire vise à concevoir une plateforme de communication simple en consultant une base de données MySQL avec PHP.
